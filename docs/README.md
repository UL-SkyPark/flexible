# 근무표 모바일 PWA

이 폴더는 GitHub Pages 배포용 정적 웹앱입니다.

## GitHub Pages 설정

1. GitHub 저장소에 이 프로젝트를 push합니다.
2. 저장소 `Settings` → `Pages`로 이동합니다.
3. `Build and deployment`의 Source를 `Deploy from a branch`로 선택합니다.
4. Branch는 `main`, Folder는 `/docs`를 선택합니다.
5. 저장 후 표시되는 GitHub Pages 주소로 접속합니다.

## 사용 방식

- 구글시트 근무표를 읽어 직원별 월간 달력을 표시합니다.
- 모바일 웹앱은 보기 전용입니다.
- 근무자 선택, 월 이동, 새로고침, 날짜별 근무 코드 확인만 남긴 가벼운 확인용 화면입니다.
- 날짜를 누르면 근무 코드와 메모를 모달로 확인합니다.
- iPhone/iPad는 Safari 공유 버튼 → `홈 화면에 추가`로 앱처럼 사용할 수 있습니다.
- Android는 Chrome 메뉴 → `홈 화면에 추가` 또는 설치 안내 버튼으로 추가할 수 있습니다.
